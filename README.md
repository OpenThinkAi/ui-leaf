# ui-leaf

Bring your own view to any CLI. Local, customizable, language-neutral.

> **Status:** placeholder release to reserve the package name. Real work in progress.

## What this will be

A small runtime that lets any CLI mount a customizable browser view on demand. The CLI pipes structured data in; the view renders it; mutations flow back through the CLI — never directly to the backing API.

Designed to work for CLIs in any language (Node, Bun, Rust, Python, Bash) via a language-neutral protocol.

Track progress at https://github.com/OpenThinkAi/ui-leaf
